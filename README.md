# fresh-toolkit
# 简洁工具箱 · 只为真正需要的功能
# 项目初衷
之前用过很多工具类应用，要么功能单一、需要装好几个才能满足日常需求；

要么大而臃肿，塞满用不上的功能和各种广告，真正顺手的功能却没几个；

我偏爱简洁，所以自己动手把真正会用到的功能集合到一起 --- 没有多余的花哨，没有广告打扰，打开即用；

因为只有安卓系统手机，其他手机系统不好验证，仅支持安卓系统；

此项目仅用于方便大家使用，禁止用于商业用途。


# 特点
功能聚合：将日常高频小工具集成于一处，告别多应用切换。

极致简洁：只保留自己验证过、确实会用到的功能，不堆砌。

零广告：没有任何广告，不分散注意力。

使用流畅：轻量设计，启动快速，操作直观。
